
# Accredian-Backend 


This is the backend of Accredian application.



## Local setup

- Copy .env.example to .env.

```bash
  cp .env.example .env
```

- Clone the repo.

```bash
   git clone https://github.com/<yourusername>/accredian-backend .git
```

- Install dependencies

```bash
  npm install 
```

- Start the server.
```bash
  npm run dev
```
